## Guide d'utilisation du projet

### Avec les sources .py

1. Cloner le dépôt
   ```
   git clone git@github.com:Oscosc/Projet-TSAI-2025.git
   ```
2. Se placer dans `src` et exécuter le programme principal :
   ```
   python3 run_script.py
   ```
   Les résultats s'affichent directement à la fin de l'exécution.
3. Pour visualiser à nouveau les résultats, utiliser le programme de lecture :
   ```
   python3 read_data.py
   ```

### Avec les notebooks .ipynb

1. Cloner le dépôt
   ```
   git clone git@github.com:Oscosc/Projet-TSAI-2025.git
   ```
2. Ouvrir le fichier `run_notebook.ipynb` et exécuter toutes les cellules, sauf celles qui permettent de télécharger à nouveau les données et le dépôt.
   Si vous souhaitez utiliser un environnement type Google Colab, récupérer seulement le notebook et exécuter toutes les cellules.
